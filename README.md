# Device Inspection Tool

## Overview
Interactive CLI tool that simulates connecting to a device, sending CLI-style commands, parsing raw output, and building a structured device profile.

## Features
- Interactive terminal shell
- Mock device connection
- Command execution
- Raw output parsing
- Device profile summary
- Connection state handling

## Commands
connect
disconnect
connectionstatus
checkbattery
checkos
checkmodel
showprofile
exit

## How to Run
python main.py
